Welcome to Griffon!

This will install and setup Griffon [http://griffon.codehaus.org].

Griffon is a Grails like application framework for developing desktop applications in Groovy. It leverages the convention-over-configuration paradigm and lessons learned by the Grails developers & community, applying them to desktop application development.
